Clipsjni: Clips Java Native Interface

Installation

  1. download from CLIPS
  2. unzip
  3. put clipsjni.dll in windows system32 dir
  4. set classpath in JServ: wrapper.classpath=.../clipsjni.jar
  5. in java program load the clipsjni.dll dynamically: System.load(clipsjni.dll);

Examples come with the package

  1. AnimalDemo (animaldemo.clp) (bcdemo.clp)
  2. AutoDemo (autodemo.clp)
  3. SudokuDemo (sudoku.clp) (solve.clp)
  4. WineDemo (winedemo.clp)
