學期報告格式

  1. 題目
    姓名, 學號, 系別, 年級, 電話
  2. 摘要
  3. 本文
    內容建議包括(格式不拘):
    1. 簡介
    2. 背景資料說明
      • 文獻的survey
      • 研究動機
      • 問題敘述
      • 解法對策說明
    3. 問題說明
    4. 主文（如解法或系統說明）
    5. 結論或綜合說明
  4. 參考資料
    格式如下:
    • (書) B. Ezzell, Using Turbo Pascal 6.0, Addison-Wesley, 1991.
    • (論文) A. Lempel, "Cryptology in Transition", Computing Surveys, vol.11, no.4, pp.285-303, Dec., 1979.
  5. 附錄
    1. 原始程式或知識庫列印
    2. 諮詢系統或執行程式的結果
    3. 其他
